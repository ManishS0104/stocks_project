# 📈 Stock Analytics Dashboard FY2025-26

![Dashboard Preview](dashboard-preview.png)

## 🚀 Project Overview

This project is a fully interactive **Stock Analytics Dashboard** developed using:

- 📊 Excel Dashboards
- 📈 Interactive HTML Visualizations
- 📉 Business Analytics
- 📋 Data Cleaning & Reporting

The dashboard provides deep insights into:

- Sales Performance
- Product Revenue Analysis
- Payment Mode Tracking
- Monthly Transactions
- Pending Collections
- Labour & Purchase Expenses
- Product Performance Analytics

---

## 📁 Project Files

| File Name | Description |
|------------|-------------|
| `Stock_FY2025-26_ExcelDashboard.xlsx` | Main Excel dashboard file |
| `Stock_2024-25_Dashboard.xlsx` | Previous financial year dashboard |
| `Stock_2024-25_Cleaned_Dashboard.xlsx` | Cleaned dataset for FY 2024-25 |
| `Stock_FY2025-26_Cleaned_Dashboard.xlsx` | Cleaned dataset for FY 2025-26 |
| `Stock_FY2025-26_Excel_Dashboard.xlsx` | Advanced Excel dashboard |
| `Stock_FY2025-26_Dashboard.html` | Interactive HTML dashboard |
| `Interactive_Stock_Dashboard.html` | Interactive Plotly dashboard |
